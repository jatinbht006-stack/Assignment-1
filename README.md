# SkillPath Website Enhancement

A responsive, multi-page React application built for the SkillPath assignment, focusing on component-based architecture and client-side routing.

## Features
- **Responsive Layout**: Modular Navbar and Footer components.
- **Dynamic Routing**: Navigating via React Router (v6).
- **Interactive Forms**: Controlled form with state management and validation.
- **Error Handling**: Custom 404 page for invalid routes.

## Tech Stack
- React.js
- React Router DOM
- Parcel (Bundler)
